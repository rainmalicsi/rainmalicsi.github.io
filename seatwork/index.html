Six (6) ways to display an italicized text

This is italicized using <i></i>. The italics element is a physical element for displaying italicized text

This is italicized using emphasis <em></em> . The emphasis element is a logical element for emphasizing important portions of a document

This is italicized using citation <cite></cite>. The citation element identifies a portion of your document as a reference to an outside source.

This is italicized using variable <var></var>. The logical element indicates a variable, as might be used in computer code.

This is italicized using dfn <dfn></dfn>. This logical element identifies a portion of text as defining instance of a term

This is italicized using address <address></address>. Defines contact information for the author/owner of a document


Two (2) ways to display boldface

This is bold using physical element b <b></b>. The bold element is a physical element that allows you to render boldface text.

This is bold using strong <strong></strong>. The strong also displays in bold. This logical element indicates a heavier or stronger emphasis than does , but there is usually no difference in how the two look in the browser

Three (3) ways to display a monospace text

This is displaying a monospace text using <code></code>. This logical element not only displays in a courier, fixed width-font font, but also indicates that the text is a portion of computer code.

Keyboard input using kbd <kbd></kbd>. Standing for keyboard, this logical element identifies its contents as keyboard input.

Another logical element, using samp <samp></samp>, describes as sample output. This is another logical element. It describe its content as sample output, most often rendering text in monospaced font.


Two (2) ways to display strikethrough

This physical element will display text as “strikethrough” using <s></s>.

The deleted element is a logical element that indicates that the text has been deleted but left in the page. This uses <del></del>


Two (2) ways to display underlined text

The underline element will render text with a line underneath. This uses <u></u>

The inserted text element is a logical element that indicates text that has been inserted since the document was written. This uses <ins></ins>
